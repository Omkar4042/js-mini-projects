# JavaScript Mini Projects

A collection of small JavaScript projects built to practice core frontend development concepts such as DOM manipulation, event handling, user interaction, and dynamic UI updates using **HTML, CSS, and Vanilla JavaScript**.

Each project focuses on a specific concept and is organized into separate folders with its own files and live demo.

---

## 🚀 Projects Included

### 1️⃣ BMI Calculator

A simple tool that calculates Body Mass Index based on the user's height and weight input.

**Features**

* Input validation
* Real-time BMI calculation
* Displays BMI category (Underweight, Normal, Overweight)
* Simple and clean UI

**Live Demo**
https://omkar4042.github.io/js-mini-projects/bmi-calculator

---

### 2️⃣ Color Changer

An interactive page that changes the background color when different color boxes are clicked.

**Features**

* DOM event handling
* Dynamic background color updates
* Simple UI interaction

**Live Demo**
https://omkar4042.github.io/js-mini-projects/color-changer

---

### 3️⃣ Random Color Generator

Generates random background colors automatically when the user clicks the start button and stops when the stop button is pressed.

**Features**

* Random color generation
* `setInterval` and `clearInterval`
* Dynamic UI updates

**Live Demo**
https://omkar4042.github.io/js-mini-projects/random-color-generator
